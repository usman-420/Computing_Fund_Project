Project Overview

This project demonstrates the setup of a complete Linux-based environment using VirtualBox and Ansible automation. The environment was configured as part of Module 13 – Put up an environment.

Virtual Machines

  • Jump Server (jumpserver1) – Acts as the control node – Runs Ansible – Provides VNC access with a desktop environment
  • Web Server (webserver1) – Hosts an Apache web server – Serves this website automatically via Ansible

Technologies Used

  • Ubuntu Linux 24.04
  • Oracle VirtualBox
  • Ansible (automation & configuration management)
  • Apache Web Server
  • SSH key-based authentication
  • Git & GitHub

Automation Details

Ansible playbooks were used to:

  • Create and configure users
  • Install required packages automatically
  • Configure VNC on the jump server
  • Install and enable Apache on the web server
  • Deploy this website directly from GitHub

Conclusion

This setup demonstrates how infrastructure and services can be deployed reliably and repeatedly using automation tools, following best practices for modern system administration.
